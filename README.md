# download-materi-lms

## Installasi
  * Python3 <br>
  * Google Chrome<br>
  
  Windows
  ```
  python -m venv env
  env\Scripts\activate
  (env)> pip install -r requirements.txt 
  ```

  Linux
  ```
  python3 -m venv env
  source env\bin\activate
  (env)> pip install -r requirements.txt 
  ```
